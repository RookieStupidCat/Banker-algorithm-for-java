# 银行家算法

#### 项目介绍
该项目是在学习计算机操作系统的时候针对进程的避免死锁做的一个实现，用户输入系统的各项临界资源量和进程数以及各进程的初始已占用资源，选择一个进程进行资源申请，然后使用 银行家算法 判定是否通过，以及此时是否有安全序列
